# AI Model Icons

Reusable SVG icons for Xiaohei motion scenes.

| ID | File | Usage |
|---|---|---|
| `chatgpt` | `chatgpt.svg` | OpenAI / ChatGPT model card icon |
| `claude` | `claude.svg` | Anthropic Claude model card icon |
| `flash-lightning` | `flash-lightning.svg` | Flash / speed / lightweight execution model icon |
| `stepfun` | `stepfun.svg` | Step / Stepfun model card icon |
| `deepseek` | `deepseek.svg` | DeepSeek model card icon |
| `qwen` | `qwen.svg` | Qwen model card icon |

Use the root `../index.json` as the primary machine-readable source. This folder's `index.json` is a model-icon subset for quick lookup.
