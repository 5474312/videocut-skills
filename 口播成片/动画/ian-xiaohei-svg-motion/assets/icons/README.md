# Icon Library

Reusable SVG icons for Xiaohei motion scenes.

Use `index.json` as the source of truth. Store original SVG files in subfolders and inline them into scene SVG during generation.

## Folders

| Folder | Purpose |
|---|---|
| `ai-models/` | AI model and provider logos |
| `common/` | Generic task and interface icons |

## Sources

| Source | Use |
|---|---|
| User-provided SVG | ChatGPT, Claude, Flash lightning |
| `@lobehub/icons-static-svg` | Stepfun, DeepSeek, Qwen model logos |
| `lucide-static` | Common task/interface icons such as bug, database, globe, gauge, brain, copy, trending-down |
