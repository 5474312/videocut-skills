# xiaohei-series-reference

小黑 SVG 动效系列参考页。目标是提供多种口播解释画面的可复制模板，而不是只保留单张“两个断点”样张。

## 文件清单

| 文件 | 地位 | 功能 |
|------|------|------|
| `index.html` | 核心 | 四个小黑漫画感 SVG/GSAP 动效模板 |
| `preview.png` | 预览 | 静态审核截图 |
| `preview-mid.png` | 预览 | 播放中段截图，用于检查递进和重叠 |
| `vendor/gsap.min.js` | 依赖 | 本地 GSAP runtime |

## 模板系列

| 模板 | 适合场景 |
|------|----------|
| `stuck-input` | 素材、需求、信息进入系统前卡住 |
| `filter-funnel` | 筛掉无效信息，保留可用素材 |
| `bridge-handoff` | 内容观点到行动入口之间缺承接 |
| `feedback-loop` | 审核发现问题，回到上一步修正后再输出 |

## 维护规则

- 每张图只讲一个动作或隐喻。
- 小黑必须承担核心动作，不能只是站在旁边。
- 批注必须分阶段显示，不要一开始全部出现。
- 批注避开角色、物体运动路径和最终落点。
- 不使用透明动线、ghost path、进度感箭头或 `motionGuide`。
- 流向优先由物体本身移动表达。
- 每个模板都要支持 `?static=1` 静态审核。
- 每个模板都要支持 `?frame=mid` 固定中间帧审核，用来检查递进、遮挡和文字抢跑。
